A deep learning project that uses a custom Convolutional Neural Network (CNN) to classify plant leaf diseases from images.
The model is trained on the New Plant Diseases Dataset (Augmented) and can identify 38 different plant disease classes.

Tech Stack

Python
TensorFlow / Keras
NumPy
Matplotlib
Scikit-learn
OpenCV

Dataset

Source: Kaggle
Name: New Plant Diseases Dataset (Augmented)
Classes: 38

Features

Image augmentation
Multi-class classification
Accuracy & loss visualization
Confusion matrix & classification report
Single image prediction with confidence score
